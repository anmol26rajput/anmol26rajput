### 🚀 Projects

**Web apps**
- [**Sarva**](https://github.com/anmol26rajput/Sarva) — Privacy-focused tool to upload, edit, and clean CSV/PDF/DOCX/PPTX files. Django backend + Pandas for processing, vanilla JS frontend, includes bulk row splitting and AI-assisted data clean-up.
- [**Mindful**](https://github.com/anmol26rajput/Mindful) — Habit and goal tracker with a daily journal, streak tracking, and a personal dashboard. Built on Django with SQLite/PostgreSQL.
- [**G20NewsHub**](https://github.com/anmol26rajput/G20NewsHub) — React news aggregator for G20-related coverage, with a companion Chrome extension that surfaces top headlines in a popup. Pulls from the Algolia API.
- [**Simple-Poll-App**](https://github.com/anmol26rajput/Simple-Poll-App) — Django-based polling app (classic polls tutorial structure, extended).
- [**Django-Authentication**](https://github.com/anmol26rajput/Django-Authentication) — Auth system built in Django — login, signup, session handling.

**Machine Learning / NLP**
- [**Django-ML-App**](https://github.com/anmol26rajput/Django-ML-App) — Serves predictions from a Random Forest model (trained on the Iris dataset) through both a web form and a JSON REST API. Includes unit tests.
- [**SMS-Spam-Detection**](https://github.com/anmol26rajput/SMS-Spam-Detection) — NLP pipeline (NLTK preprocessing + bag-of-words/TF-IDF) feeding a Naive Bayes classifier to flag spam SMS messages.
- [**House-Price-Prediction**](https://github.com/anmol26rajput/House-Price-Prediction) — Regression model for predicting house prices from structured data.
- [**minchu.ipynb**](https://github.com/anmol26rajput/minchu.ipynb) — Python-based voice assistant notebook.

**Utilities & misc**
- [**Text-To-Speech-Speech-To-Text-Conversion**](https://github.com/anmol26rajput/Text-To-Speech-Speech-To-Text-Conversion) — Two-way audio/text conversion using gTTS and SpeechRecognition, with multi-language support.
- [**Java-projects**](https://github.com/anmol26rajput/Java-projects) — Small Java programs (ATM machine simulation and others).
- [**CBTCIP**](https://github.com/anmol26rajput/CBTCIP) — Internship projects for Cipherbyte — includes a Mastermind game and Rock-Paper-Scissors in Python.
